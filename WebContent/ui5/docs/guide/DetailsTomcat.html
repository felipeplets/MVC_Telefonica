Getting Started - Download and Setup - Tomcat Setup

Setting up Tomcat with SAPUI5 involves 3 steps:

Download Tomcat

  • Download Tomcat from the appropriate website
  • Either install the Windows Service Installer or download and unzip the zip file (you can find the startup script in the bin folder)
  • If you run Tomcat without Eclipse, make sure, you have set the environment variable JAVA_HOME, pointing to a JDK7.

Deploy SAPUI5 on Tomcat without Eclipse

  • You could setup a Tomcat user with "manager" privileges (see the Tomcat documentation) and use the tomcat-manager application for deployment or
  • you could (this is recommended) just move your war file into the webapps folder inside the tomcat directory. Tomcat will take care of the deployment, but a restart might be necessary.

Configure Tomcat within Eclipse

You need to register an Tomcat Web Server in order to run Web applications within Eclipse:

  • Open the Java EE perspective in Eclipse
  • Open the Servers view (Hint: CTRL+3 - then enter Servers)
  • Use context menu -> New -> Server to open the new server wizard
  • Choose Apache -> Tomcat 7.0 and add a new server runtime by pointing to the folder where you unzipped the Tomcat into
  • When you finish the wizard you should see your Tomcat v7.0 server in the Servers view
  • Finally open the configuration overview by double click on the Tomcat v7.0 server in the Servers view. There you need to select the server option: Serve modules without publishing
